
НАСТРОЙКА "ГОРЯЧИХ" КЛАВИШ И РЕДАКТИРОВАНИЕ ПОДСКАЗОК
---------------------------------------------------------

После установки данного обновления пользователь получает возможность изменять "горячие" клавиши, используемые в игре по умолчанию, а также редактировать текст всплывающих подсказок, появляющихся на экране во время игры. Для этого в папке с установленной игрой необходимо создать файл под названием "CustomKeys.txt". Клавиши, указанные в этом файле, будут использоваться в игре вместо клавиш по умолчанию. Для создания и редактирования текстового файла можно использовать любой текстовый редактор (например, программу "Блокнот"). После создания файла нужно запустить игру и включить соответствующий параметр в меню настройки.

ВНИМАНИЕ: компания Blizzard Entertainment не предоставляет техническую поддержку по вопросам, связанным с использованием файла "CustomKeys.txt" и данных, содержащихся в нем. Компания Blizzard Entertainment не обязана и не будет предоставлять техническую поддержку по любым сторонним специализированным программам, предназначенным для редактирования файла "CustomKeys.txt", более того, компания не рекомендует пользователям использовать такие программы. Любые изменения, которые вносятся в файл "CustomKeys.txt" (в том числе с использованием сторонних программ), осуществляются пользователем исключительно на свой страх и риск.


НАСТРОЙКА "ГОРЯЧИХ" КЛАВИШ
------------------------

"Горячие" клавиши настраиваются следующим образом: в тексте файла, в квадратных скобках указывается индекс, соответствующий боевой единице или зданию. В следующей строке слева от знака равенства должен находиться один из нижеследующих идентификаторов:
- Hotkey (клавиша начала действия)
- Unhotkey (клавиша прекращения действия)
- Researchhotkey (клавиша исследования)

Справа от знака равенства должен находиться символ, соответствующий "горячей" клавише (заглавная буква). Например, чтобы тренировке бугая соответствовала клавиша 'D', файл должен содержать следующие строки:

[ogru]
Hotkey=D

Иногда одному действию соответствует целая последовательность "горячих" клавиш. Например, каждому из трех уровней усовершенствования оружия ближнего боя (войска Альянса) может соответствовать своя "горячая" клавиша. Чтобы это усовершенствование проводилось при нажатии клавиш 'X', 'Y' и 'Z' (уровни 1, 2 и 3 соответственно), файл должен содержать следующие строки:

[Rhme]
Hotkey=X,Y,Z

Для некоторых действий можно назначить клавиши их начала и окончания. Например, чтобы пехотинец Альянса укрывался за щитом при нажатии клавиши 'A' и убирал щит при нажатии клавиши 'B', файл должен содержать следующие строки:

[Adef]
Hotkey=A
Unhotkey=B

Кроме того, можно изменить "горячие" клавиши, соответствующие освоению способностей и проведению усовершенствований. Например, чтобы герой-паладин освоил способность "Воскрешение" при нажатии клавиши 'U', файл должен содержать следующие строки:

[AHre]
Researchhotkey=U

Присвоение одной "горячей" клавиши сразу нескольким командам не влияет на стабильность игры, однако при этом будет действовать только одна из назначенных клавиш. Учтите, многие навыки используются сразу несколькими родами войск. Даже если "горячая" клавиша нормально работает с одной боевой единицей, с другой боевой единицей она может не действовать.


ИЗМЕНЕНИЕ ПОЗИЦИЙ КНОПОК
---------------------------------

Изменение позиций кнопок на панели управления войсками осуществляется следующим образом: в тексте файла в квадратных скобках указывается индекс, соответствующий этой кнопке. В следующей строке слева от знака равенства должен находиться один из нижеследующих идентификаторов:
 - Buttonpos (кнопка начала действия)
 - Unbuttonpos (кнопка прекращения действия)
 - Researchbuttonpos (кнопка исследования)

Справа от знака равенства должна быть указана позиция кнопки: номер столбца, считая слева (от 0 до 3), и номер строки, считая сверху (от 0 до 2). Например, чтобы кнопка тренировки бугая находилась в крайнем справа столбце средней строки, файл должен содержать следующий текст:

[ogru]
Buttonpos=3,1

При совпадении позиций нескольких кнопок у одной и той же боевой единицы конфликтующие кнопки будут автоматически переставлены на другие позиции. Если переставить кнопку не удастся, она останется недоступной. Учтите: некоторые возможности присущи сразу многим боевым единицам или зданиям. Даже если кнопка нормально выводится на экран при выборе одной боевой единицы, при выборе другой боевой единицы ее позиция может быть занята.


РЕДАКТИРОВАНИЕ ТЕКСТА ПОДСКАЗОК
--------------------------

Подсказки, появляющиеся на экране во время игры, можно отредактировать и привести в соответствие новым "горячим" клавишам. Для этого в тексте файла в квадратных скобках укажите индекс подсказки. В следующей строке слева от знака равенства должен находиться один из нижеследующих идентификаторов:
 - Tip (подсказка для действия)
 - Untip (подсказка для отмены действия)
 - Researchtip (подсказка для исследования)
 - Revivetip (подсказка для воскрешения героя у алтаря)
 - Awakentip (подсказка для воскрешения героя в таверне)

Справа от знака равенства должен находиться текст подсказки, например:

[ogru]
Hotkey=T
Tip=Тренировка бугая (T)

[Rhme]
Hotkey=X,Y,Z
Tip=Железные мечи (X), Стальные мечи (Y), Мифрильные мечи (Z)

[Adef]
Hotkey=A
Unhotkey=B
Tip=Укрыться за щитом (A)
Untip=Покинуть укрытие (B)

[AHre]
Researchhotkey=U
Researchtip=Изучить навык "Воскрешение" (U)


ЦВЕТОВОЕ ВЫДЕЛЕНИЕ
---------------------------------------

В тексте подсказок часть символов можно выделить другим цветом. Начало выделения в тексте подсказки задается следующей последовательностью символов (без кавычек): "|cARGB", где A, R, G и B – шестнадцатеричные значения альфа-канала (A), красной (R), зеленой (G) и синей (B) составляющих цвета. Окончание цветового выделения задается последовательностью символов "|r". Например, чтобы в тексте подсказки горячая клавиша "Т" была выделена желтым цветом, файл должен содержать следующий текст:

[ogru]
Hotkey=T
Tip=Тренировка бугая (|cffffcc00T|r)


Воспользуйтесь новыми возможностями, чтобы настроить управление в соответствии со своим индивидуальным стилем игры!
