НОВОСТИ

С 19-21 октября на базе танцевально-спортивного клуба Санрайз состоялись традиционные осенние сборы. Главными Организаторами мероприятия были заслуженные мастера спорта и трехкратные чемпионы мира. На протяжении трех дней команда профессиональных педагогов делилась со спортсменами своим опытом и мастерством.Такого рода подготовка является наиболее эффективной и помогает извлечь из занятий максимум пользы для танцоров.
